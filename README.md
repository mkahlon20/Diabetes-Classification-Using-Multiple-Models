# Diabetes Classification Experiment

## Project Overview
This project analyzes the diabetes and heart disease datasets using multiple machine learning classification models. The workflow includes preprocessing, PCA, SMOTE, model training, cross validation, and evaluation.

## Datasets
- diabetes.csv
- heart.csv

## Models Used
- KNN
- Gaussian Naive Bayes
- Logistic Regression
- Decision Tree
- SVM

## Setup Instructions
1. Install Python
2. Install required libraries
3. Open notebooks in VS Code or Jupyter
4. Run all cells

- secondary_dataset_analysis.ipynb
